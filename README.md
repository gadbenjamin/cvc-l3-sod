Game Title: Forest Coin Adventure

The game is called Forest Coin Adventure.

The story is about a young hero named Alex.
Alex is walking inside a green forest.

Inside the forest, there are magic golden coins.
These coins give power and life to the hero.

There is also an enemy called Shadow.
Shadow moves inside the forest and tries to stop Alex.

Each golden coin increases the score.

If Alex loses all his lives, the game ends.
This means Game Over.
